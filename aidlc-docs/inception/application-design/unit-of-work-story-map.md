# Unit of Work — Requirements Mapping

> User Stories 단계를 생략했으므로, 기능 요구사항(FR)을 유닛에 매핑합니다.

## Unit 1: Backend

| Requirement | Description |
|-------------|-------------|
| FR-1 | 테이블 태블릿 자동 로그인 (API) |
| FR-2 | 메뉴 조회 (API) |
| FR-4 | 주문 생성 (API) |
| FR-5 | 주문 내역 조회 (API) |
| FR-6 | 매장 인증 — 로그인 (API) |
| FR-7 | 실시간 주문 모니터링 (SSE API) |
| FR-8 | 테이블 관리 (API) |
| FR-9 | 메뉴 관리 CRUD + 이미지 업로드 (API) |
| FR-10 | 관리자 회원가입 (API) |
| NFR-2 | Docker Compose 배포 설정 |
