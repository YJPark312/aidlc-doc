# Requirements Document — 테이블오더 시스템

## Intent Analysis
- **User Request**: 테이블 오더 시스템 실습 프로젝트 개발
- **Request Type**: New Project (Greenfield)
- **Scope**: System-wide — 고객용 UI, 관리자 UI, 백엔드 API, 데이터베이스
- **Complexity**: Complex — 다수 기능, 실시간 통신(SSE), 인증/세션 관리, 파일 업로드

---

## Technology Stack
- **Backend**: Python + FastAPI
- **Frontend**: Vue.js
- **Database**: MySQL
- **Deployment**: 로컬 개발 환경 (Docker Compose)
- **Image Storage**: 서버 로컬 파일 시스템
- **Real-time**: Server-Sent Events (SSE)
- **Authentication**: JWT 토큰 기반

---

## Functional Requirements

### FR-1: 고객용 — 테이블 태블릿 자동 로그인
- 관리자가 매장 식별자, 테이블 번호, 비밀번호 1회 설정
- 설정 정보 로컬 저장 후 자동 로그인

### FR-2: 고객용 — 메뉴 조회
- 카테고리별 메뉴 분류 및 표시
- 메뉴 상세: 메뉴명, 가격, 설명, 이미지
- 카드 형태 레이아웃, 터치 친화적 UI (최소 44x44px)

### FR-3: 고객용 — 장바구니 관리
- 메뉴 추가/삭제, 수량 조절
- 총 금액 실시간 계산
- 클라이언트 측 로컬 저장 (새로고침 시 유지)

### FR-4: 고객용 — 주문 생성
- 장바구니 → 주문 전환
- 주문 정보: 매장ID, 테이블ID, 메뉴 목록, 총 금액, 세션ID
- 성공 시: 주문번호 표시, 장바구니 비우기, 메뉴 화면 리다이렉트
- 실패 시: 에러 메시지, 장바구니 유지

### FR-5: 고객용 — 주문 내역 조회
- 현재 테이블 세션 주문만 표시
- 주문번호, 시각, 메뉴/수량, 금액, 상태(대기중/준비중/완료)

### FR-6: 관리자용 — 매장 인증
- 매장 식별자 + 사용자명 + 비밀번호 로그인
- JWT 토큰 기반, 16시간 세션
- 비밀번호 bcrypt 해싱
- 관리자 회원가입 기능 포함

### FR-7: 관리자용 — 실시간 주문 모니터링
- SSE 기반 실시간 주문 업데이트 (2초 이내)
- 테이블별 그리드/카드 레이아웃
- 테이블 카드: 총 주문액, 최신 주문 미리보기
- 주문 상태 변경 (대기중→준비중→완료)
- 신규 주문 시각적 강조

### FR-8: 관리자용 — 테이블 관리
- 테이블 초기 설정 (번호, 비밀번호)
- 주문 삭제 (확인 팝업, 총 주문액 재계산)
- 테이블 세션 종료 (이용 완료 → 주문 이력 이동, 리셋)
- 과거 주문 내역 조회 (날짜 필터링)

### FR-9: 관리자용 — 메뉴 관리
- 메뉴 CRUD (메뉴명, 가격, 설명, 카테고리, 이미지)
- 이미지 파일 서버 로컬 업로드
- 메뉴 노출 순서 조정
- 필수 필드 및 가격 범위 검증

### FR-10: 관리자용 — 회원가입
- 매장 식별자, 사용자명, 비밀번호로 관리자 계정 생성
- 비밀번호 bcrypt 해싱 저장

## Non-Functional Requirements

### NFR-1: 성능
- 소규모 환경 (단일 매장, 테이블 10개 이하)
- SSE 주문 알림 2초 이내 전달

### NFR-2: 배포
- Docker Compose 기반 로컬 개발 환경
- 컨테이너: FastAPI 백엔드, Vue.js 프론트엔드, MySQL

### NFR-3: 보안
- 실습 프로젝트 수준 (Security Extension 미적용)
- 기본 JWT 인증 및 bcrypt 비밀번호 해싱은 유지

### NFR-4: 테스트
- Property-Based Testing 미적용
- 기본 단위 테스트 수준

---

## Exclusions (구현 제외)
- 결제 처리, PG사 연동, 영수증
- OAuth/SNS 로그인, 2FA
- 이미지 리사이징/최적화
- 푸시/SMS/이메일 알림
- 주방 시스템, 재고 관리
- 데이터 분석, 매출 리포트
- POS 연동, 배달 플랫폼 연동
- 다국어 기능
